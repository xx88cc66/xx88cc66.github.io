mysql增删改查

连接数据库
mysql -h 127.0.0.1 -u root -p

查看数据库
Show databases;

查看数据库下的表 比如查看sqli的
Use sqli;
Show tables;

新增数据库 create database xxx

删除数据库 drop database xxx;

导入数据库source+绝对路径；

查询表中数据 select * from admin（表）

查询uid为2的 用where
Select * from admin where uid=2

insert into admin (uid,name,pass) values
(‘1’,’user’,’122222’);(字符要加单引号)

删除数据 Delete from admin
where uid=2

修改数据 update admin set pass=‘123’ where uid=2；
